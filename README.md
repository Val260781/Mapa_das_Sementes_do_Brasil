# Mapa_das_Sementes_do_Brasil
O propjeto tem como objetivo principal, desenvolver uma plataforma digital colaborativa para mapeamento de árvores e sementes do artesanato brasileiro.

## ETAPAS DO DESENVOLVIMENTO ##
1. Base (Backend + Banco) → modelos, migrations, conexão PostgreSQL, estrutura do Go;
2. Autenticação → cadastro/login, roles (artesão, pesquisador, admin);
3. API de Sementes/Espécies → CRUD completo com upload de fotos;
4. Geolocalização → registro de localização das árvores no mapa;
                  ------DEPLOY------
5. Frontend Web → mapa interativo + UI para cadastro e consulta
6. Validação Colaborativa → sistema de aprovação entre usuários
7. Dashboard → métricas, estatísticas, gráficos
8. PWA → transformar em app instalável no celular

# Ferramentas utilizadas #
- Visual Studio Code - versão 1.125.0 (programação em Go)
- pgAdmin 4 (Banco de Dados)
- Github (Armazenamento dos arquivos e progressão em parceria)
- Postman (Testes da aplicação e funcionamento)
- Railway (Deploy)
- Cloudinary (armazenamento - 25GB grátis) através de URL pública
